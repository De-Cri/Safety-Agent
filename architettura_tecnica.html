Safety Agent — Mappa tecnica del progetto

Agente LLM (Gemini 2.5 Flash) per l'analisi di eventi di sicurezza CCTV via tool MCP su PostgreSQL. Passa il mouse sopra ogni nodo per i dettagli implementativi.

Stack: FastAPI · MCP (FastMCP) · SQLAlchemy · PostgreSQL · Gemini 2.5 Flash · matplotlib ~3.6k righe di codice Python Snapshot repo: 22 giugno 2026

Principio cardine

Il modello non riceve mai il dump del DB: accede solo via tool MCP. Risposte fondate sui dati, costi sotto controllo, privacy.

Risparmio token (LEAN vs FULL)

55–57% in meno di token di input a parità di qualità della risposta (LLM-as-judge). Vedi nodo T-Eval / Token Benchmark.

T-Eval — ultimo run (18/06)

R 0.91 · U 0.92 · V 0.86 su 68 casi (58 in-scope + 10 out-of-scope, refusal rate 1.0)

Punti deboli noti

get_event_by_id → V=0.0 (3 casi, da investigare). multi_tool → R=0.5 (3 casi). Vedi nodo T-Eval per dettagli.
